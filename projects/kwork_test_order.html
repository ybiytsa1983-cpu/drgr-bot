📊 Результаты тестов

🔐 Обход CAPTCHA

Интеграция с 2captcha API для автоматического решения reCAPTCHA. Selenium-браузер с антидетект-режимом.

✅ Реализовано

🤖 Подтверждение «не робот»

Автоматическое обнаружение и решение CAPTCHA на страницах. Поддержка reCAPTCHA v2/v3 и hCaptcha.

✅ Реализовано

🔑 Вход в аккаунт

Selenium-логин с автозаполнением полей. Поддержка кастомных селекторов. Headless-режим по умолчанию.

✅ Реализовано

🛑 Кнопка «Снять задачу»

Полная система управления задачами: создание, отслеживание, отмена (Снять). UI с красной кнопкой.

✅ Реализовано

🔄 Выбор количества циклов

Слайдер + числовой ввод (1-1000). Настраиваемая задержка между циклами. Прогресс в реальном времени.

✅ Реализовано

💬 AI Chat

Ollama/LM Studio интеграция. Генерация текстов, статей, 3D-моделей (Three.js), видео (moviepy).

⚠️ Нужен LLM
