📋 Как пользоваться переученной ВМ

  • Нажми Создать переученную ВМ выше — дождись завершения (1–3 мин)
  • После создания модель drgr-visor появится в списке моделей
  • Открой настройки (кнопка ☰ слева вверху) → выбери drgr-visor в поле Модель
  • Перейди на вкладку 💬+📝 Чат+Код — чат с ИИ и Monaco-редактор одновременно
  • Или перейди на 💬 Чат — там кнопки 🔍 Поиск и 📰 Статья для веб-поиска и генерации статей

ℹ️ Примечание: drgr-visor — это qwen3-vl:8b с особым системным промптом (оптимизированным для работы с Code VM, Monaco, ВИЗОРОМ и Telegram-ботом). Размер модели одинаков, так как изменяется только системный промпт, а не веса модели. Для работы требует запущенного Ollama и скачанной модели qwen3-vl:8b.

🪟 Запуск с рабочего стола (Windows)

На рабочем столе должны быть файлы — двойной клик запускает Code VM и создаёт drgr-visor:

  • ЗАПУСТИТЬ_ВМ.bat — запуск VM + автосоздание drgr-visor
  • ПЕРЕУЧИТЬ_ВМ.bat — только переучить (создать/обновить drgr-visor)
  • Code VM.lnk — основной ярлык запуска

Если значков нет на рабочем столе — вставь в PowerShell (Win+X → Windows PowerShell):

Шаг 1 — первая установка (загружает всё необходимое):

irm "https://raw.githubusercontent.com/ybiytsa1983-cpu/drgr-bot/main/run.ps1" | iex 📋

⬇ Скачать обновления (если VM уже установлена):

irm 'https://raw.githubusercontent.com/ybiytsa1983-cpu/drgr-bot/main/update.ps1' | iex 📋

Шаг 2 — после установки, запуск переученной ВМ (папка уже есть):

powershell -ExecutionPolicy Bypass -File "$env:USERPROFILE\drgr-bot\ЗАПУСТИТЬ_ВМ.ps1" 📋

Или просто запустить VM без переобучения:

powershell -ExecutionPolicy Bypass -File "$env:USERPROFILE\drgr-bot\vm.ps1" 📋

После запуска откройте браузер: http://127.0.0.1:5000
