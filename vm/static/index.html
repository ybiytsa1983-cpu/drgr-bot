🔗 Ollama URL
Если Ollama на нестандартном порту — укажите URL вручную
🤖 LM Studio URL
URL LM Studio сервера (OpenAI-совместимый API, порт 1234). Модели LM Studio отображаются в селекторе с префиксом lmstudio:
🧠 text-generation-webui URL
URL oobabooga/text-generation-webui с флагом --api. Запуск: python server.py --api --listen. Модели отображаются с префиксом tgwui:
🦘 Roo Code URL
URL бэкенда для Roo Code (OpenAI-совместимый API — тот же сервер, что использует расширение VS Code Roo Code). Модели отображаются с префиксом roo:
🤝 Open Agentic Framework URL
URL Open Agentic Framework (оркестратор агентов). Запуск: docker-compose up -d
🧊 3D Generation Service URL (TripoSR / Hunyuan3D)
Локальный 3D-генератор: TripoSR, Hunyuan3D-2 или NVIDIA 3D. Сервис должен принимать POST /generate.
🌐 AI Website Builder URL
AI-генератор сайтов: build-a-site или AI-Website-Builder. Без URL — сайт генерирует локальная ВМ.
🎬 Video Editor Service URL (omniclip / twick)
Видеоредактор-бэкенд: omniclip или twick. Без URL — ВМ генерирует EDL-сценарий.
🌐 Remote VM URL (Google Colab)
URL VM запущенной в Google Colab (через ngrok). Локальный сервер проксирует запросы к удалённой VM.
👁 Vision VM URL (Ollama с vision-моделью)
URL отдельного Ollama с vision-моделью (llava, minicpm-v, moondream2). Когда подключена — все vision-запросы роутятся сюда. Рекомендуем: ollama pull llava:7b (≈4 ГБ) или ollama pull moondream (≈1.7 ГБ). Запустить второй Ollama (PowerShell): $env:OLLAMA_HOST="http://127.0.0.1:11436"; ollama serve
🎨 Stable Diffusion URL (AUTOMATIC1111 / SD.Next)
Запустить AUTOMATIC1111: webui.bat --api (порт 7860). Генерация изображений по тексту.
⚙ ComfyUI URL
Запустить ComfyUI: python main.py (порт 8188). Резервный бэкенд для генерации изображений.
📱 Telegram Bot Token
Получить у @BotFather в Telegram. После сохранения токена бот перезапускается автоматически.
🧠 ВМ для Telegram бота
Какой ИИ-бэкенд использует бот при общении в Telegram. «Авто» — сам выбирает доступный.
🚀 Автозапуск Windows
Регистрирует задачу в Планировщике заданий Windows — VM запускается автоматически при входе в систему.
💬 Telegram Chat ID
ID чата/группы для уведомлений бота (узнать: /start у @userinfobot)
🤖 Модель Ollama / LM Studio
Модели Ollama, LM Studio (lmstudio:), text-generation-webui (tgwui:), Roo Code (roo:) — настройте URL выше
