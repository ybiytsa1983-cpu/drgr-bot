🧠 Психокоррекция

Сервер активен VM Панель GitHub

📥 Команда для установки и запуска

Откройте PowerShell на Windows и вставьте эту команду:

PowerShell (одна строка) [Net.ServicePointManager]::SecurityProtocol=[Net.SecurityProtocolType]::Tls12; irm https://raw.githubusercontent.com/ybiytsa1983-cpu/drgr-bot/main/start_vm.ps1 | iex

Или через Git:

Git + Python git clone https://github.com/ybiytsa1983-cpu/drgr-bot.git && cd drgr-bot && pip install -r requirements.txt && python vm/server.py

После запуска откройте: http://localhost:5005/psycho

🔬 Возможности платформы

🧠 FER-анализ

Анализ лицевой экспрессии: распознавание эмоций, валентность, возбуждение, уровень стресса. Использует нейросетевую модель для определения 7 базовых эмоций по Экману.

👁️ Анализ зрачков

Пупиллометрия: дилатация зрачков, анизокория, когнитивная нагрузка. Каскад Хаара + сегментация контуров. Скользящее среднее по 5 кадрам.

👀 Возраст по глазам

5 параметров: морщины (30%), мешки (20%), склера (15%), радужка (20%), птоз (15%). Диапазон оценки: 15-80 лет.

📊 Совокупная оценка

Стресс-балл: FER (40%) + зрачки (30%) + baseline (30%). Итоговый JSON-отчёт с рекомендациями из базы знаний.

📚 База знаний

18 научных источников, 10 методик психокоррекции: дыхание, PMR, заземление, когнитивная переоценка, журнал эмоций, рисование эмоций.

🔒 Этика и согласие

Проверка возраста 21+, информированное согласие на видеоанализ, отказ от ответственности. Без диагнозов -- только рекомендации.

🔬 Совокупная оценка

Запуск комплексного анализа: FER-эмоции, зрачки, базовый уровень стресса.

👁️ Анализ зрачков

Пупиллометрия: дилатация, анизокория, когнитивная нагрузка.

👀 Оценка возраста по глазам

Периорбитальный анализ: морщины, мешки, склера, радужка, птоз.
