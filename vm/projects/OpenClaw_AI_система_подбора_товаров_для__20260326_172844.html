OpenClaw — AI-система подбора товаров для маркетплейсов (Telegram, PostgreSQL, Docker, Wildberries, AI)

🛠 Технологический стек

  • aiogram 3.x (Telegram Bot)
  • PostgreSQL 15 + SQLAlchemy + Alembic
  • Docker Compose
  • Wildberries API / MPStats / Alibaba scraper
  • OpenClaw / Ollama / HuggingFace Inference API

📁 Структура проекта

OpenClaw_AI_система_подбора_товаров_для_/
├── docker-compose.yml
├── .env.example
├── README.md
├── bot/
│   ├── Dockerfile
│   ├── requirements.txt
│   └── main.py           # Telegram-бот (aiogram)
├── app/
│   ├── Dockerfile
│   ├── requirements.txt

├── workers/
│   ├── scraper.py        # Wildberries / Alibaba / MPStats
│   ├── analytics.py      # Юнит-экономика, скоринг товаров
│   └── scheduler.py      # APScheduler: фоновые задачи
├── db/
│   ├── models.py
│   └── migrations/       # Alembic
└── dashboard/
    └── index.html        # Web-интерфейс / дашборд

⚙️ Ключевые модули

  • scraper.py — асинхронный сбор данных (Wildberries, Alibaba, MPStats)
  • analytics.py — скоринг товаров: маржа, конкуренция, динамика спроса
  • bot/main.py — приём текстовых и голосовых команд через Telegram
  • ai_agent.py — AI-ассистент (OpenClaw / Ollama) для рекомендаций
  • db/models.py — ORM-модели: Product, Category, Supplier, Analysis
  • dashboard/index.html — дашборд: графики, фильтры, карточки товаров

🚀 Запуск

# 1. Клонировать и настроить окружение
cp .env.example .env
# Заполните .env: BOT_TOKEN, DATABASE_URL, WB_API_KEY и т.д.

# 2. Запустить через Docker Compose
docker compose up --build -d

# 3. Применить миграции БД
docker compose exec api alembic upgrade head

# 4. Открыть дашборд
# http://localhost:8080
