# drgr-bot

Telegram-бот с веб-интерфейсом и VM-сервером для управления AI-моделями.

---

## 🚀 Быстрый старт (первая установка)

> **Папка пропала с Рабочего стола?** Выполните шаги ниже — всё восстановится за пару минут.

### Шаг 1 — Установите зависимости (один раз)

| Программа | Зачем | Ссылка |
|-----------|-------|--------|
| **Python 3.10+** | Запуск бота и сервера | https://www.python.org/downloads/ |
| **Git** | Клонирование и обновление репозитория | https://git-scm.com/download/win |

> ⚠️ При установке Python обязательно отметьте **"Add Python to PATH"**.

### Шаг 2 — Скачайте скрипт установки

Скачайте **один файл** прямо по ссылке (кликните правой кнопкой → «Сохранить как»):

- **[install.bat](https://github.com/ybiytsa1983-cpu/drgr-bot/raw/main/install.bat)** ← ✅ всегда работает (ASCII-имя)
- Или `УСТАНОВИТЬ.bat` если кириллические имена отображаются корректно

Сохраните файл в любое место (например, на Рабочий стол).

### Шаг 3 — Запустите install.bat

Дважды кликните по скачанному файлу. Скрипт автоматически:

1. Клонирует репозиторий в папку `Рабочий стол\drgr-bot`
2. Установит все зависимости Python
3. Попросит ввести токен Telegram-бота (один раз)
4. Предложит сразу запустить бота

---

## ▶️ Запуск бота

После установки используйте файл в папке проекта:

```
Рабочий стол\drgr-bot\start.bat
```

Скрипт запустит два окна:
- **DRGR VM Server** — локальный сервер (`http://localhost:5000`)
- **DRGR Telegram Bot** — Telegram-бот

### Альтернативный запуск через PowerShell

```powershell
# Открыть PowerShell в папке drgr-bot, затем:
.\run.ps1
```

---

## 🔄 Обновление

Для получения новой версии запустите:

```
Рабочий стол\drgr-bot\update.bat
```

Скрипт обновит код, установит новые зависимости и предложит откатиться, если что-то пошло не так.

### Через PowerShell вручную

```powershell
.\update.ps1
# или без перезапуска бота:
.\update.ps1 -SkipRestart
```

## ⚙️ Ручная установка

Если скрипт не запускается, выполните в командной строке:

```bat
:: Клонировать репозиторий на Рабочий стол
git clone https://github.com/ybiytsa1983-cpu/drgr-bot.git %USERPROFILE%\Desktop\drgr-bot

:: Перейти в папку
cd %USERPROFILE%\Desktop\drgr-bot

:: Установить зависимости
pip install -r requirements.txt

:: Создать файл .env с токеном бота
echo BOT_TOKEN=ВАШ_ТОКЕН_ЗДЕСЬ > .env
```

Затем запустите `start.bat`.

---

## 🔑 Файл .env

Файл `.env` должен находиться в папке `drgr-bot` и содержать токен бота:

```
BOT_TOKEN=1234567890:AABBccDDeeFFggHHiiJJkkLLmmNNooPP
```

Получить токен: откройте Telegram → найдите **@BotFather** → `/newbot`.

## 📁 Структура проекта

```
drgr-bot/
├── bot.py                 # Telegram-бот
├── vm/
│   ├── server.py          # VM-сервер (Flask)
│   └── static/
│       └── index.html     # Веб-интерфейс
├── requirements.txt       # Зависимости Python
├── .env                   # Токен бота (не в репозитории!)
├── install.bat            # ✅ Первичная установка (ASCII, всегда работает)
├── start.bat              # ✅ Запуск бота + VM-сервера (ASCII, всегда работает)
├── update.bat             # ✅ Обновление до последней версии (ASCII, всегда работает)
├── run.ps1                # ✅ Запуск через PowerShell (ASCII, всегда работает)
├── update.ps1             # PowerShell-скрипт обновления
├── УСТАНОВИТЬ.bat         # Первичная установка (кириллическое имя)
├── ЗАПУСТИТЬ_БОТА.bat     # Запуск бота и VM-сервера (кириллическое имя)
└── ОБНОВИТЬ.bat           # Обновление до последней версии (кириллическое имя)
```

> 💡 **Почему два набора файлов?** Windows встроенный ZIP-архиватор иногда не извлекает файлы с кириллическими именами. Файлы `install.bat`, `start.bat`, `update.bat` и `run.ps1` — это ASCII-варианты, которые **всегда работают** независимо от способа скачивания.

## ❓ Решение проблем

| Проблема | Решение |
|----------|---------|
| `run.ps1` не найден после скачивания ZIP | Скачайте напрямую: https://github.com/ybiytsa1983-cpu/drgr-bot/raw/main/run.ps1 |
| Файлы с кириллическими именами не извлеклись из ZIP | Используйте `start.bat` или `run.ps1` — они всегда есть |
| **Папка drgr-bot пропала с Рабочего стола** | Скачайте `install.bat` и запустите его — он восстановит всё заново |
| "Python не найден" | Установите Python с https://www.python.org/downloads/ (отметьте "Add Python to PATH") |
| "Git не найден" | Установите Git с https://git-scm.com/download/win |
| Бот не отвечает в Telegram | Проверьте, что токен в `.env` правильный и что окно "DRGR Telegram Bot" открыто |
| VM-сервер недоступен (http://localhost:5000) | Проверьте, что окно "DRGR VM Server" открыто. Если нет — запустите `start.bat` снова |
| PowerShell не запускает .ps1 | Выполните один раз: `Set-ExecutionPolicy -Scope CurrentUser RemoteSigned` |
