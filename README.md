# drgr-bot

Telegram-бот с веб-интерфейсом и VM-сервером для управления AI-моделями.

## Возможности

- **💬 Чат с AI** — общение через Ollama / LM Studio (локальные LLM)
- **📝 Генератор статей** — DDG поиск + скрейпинг + LLM → HTML-статья с источниками
- **✍️ Генератор текста** — произвольные промпты → LLM
- **🤖 Telegram бот** — управление из веб-интерфейса (запуск/остановка/лог)
- **🎬 3D / Видео** — генерация Three.js и moviepy кода через LLM
- **📄 Редактор** — Monaco Editor с подсветкой синтаксиса
- **🔌 Chrome расширение** — быстрый доступ к статусу VM
- **⚙️ Настройки** — редактирование .env через веб-интерфейс

## 🚀 Быстрый старт

### Вариант 1 — PowerShell (рекомендуется)

```powershell
powershell -ExecutionPolicy Bypass -File start.ps1
```

Скрипт автоматически:
1. Проверит Python 3.10+
2. Установит зависимости
3. Обнаружит и запустит Ollama (если установлена)
4. Проверит свободность порта
5. Запустит VM сервер на `http://localhost:5002`
6. Откроет браузер

### Вариант 2 — BAT файл

```
ЗАПУСТИТЬ_БОТА.bat
```

### Вариант 3 — Ручной запуск

```bash
pip install -r requirements.txt
python vm/server.py
```

## 🌍 Доступ по сети (LAN / Интернет)

Сервер слушает на `0.0.0.0` — это значит, что он доступен с **любого устройства в вашей сети**.

### Шаг 1 — Запустите сервер на ноутбуке

```powershell
powershell -ExecutionPolicy Bypass -File start.ps1
```

Или:

```bash
pip install -r requirements.txt
python vm/server.py
```

### Шаг 2 — Узнайте IP-адрес ноутбука

**Windows:**
```
ipconfig
```
Найдите строку `IPv4 Address` (например `192.168.1.42`).

**Mac / Linux:**
```
ip addr show   # или: ifconfig
```

### Шаг 3 — Откройте с другого устройства

На другом ноутбуке/ПК/телефоне в браузере:

```
http://192.168.1.42:5002
```

> 📌 Замените `192.168.1.42` на реальный IP из шага 2, а `5002` — на ваш порт.

### Смена порта

```powershell
$env:DRGR_PORT=8080; python vm/server.py
```

Или (Linux/Mac):

```bash
DRGR_PORT=8080 python vm/server.py
```

### Доступ через Интернет (за пределами домашней сети)

**Вариант A — ngrok (быстро, бесплатно):**

```bash
# Установите ngrok: https://ngrok.com/download
ngrok http 5002
```

Вы получите URL вида `https://xxxx.ngrok-free.app` — его можно открыть с любого устройства.

**Вариант B — Проброс портов (роутер):**

1. Зайдите в настройки роутера (обычно `192.168.1.1`)
2. Найдите «Port Forwarding» / «Проброс портов»
3. Добавьте правило: внешний порт `5002` → IP ноутбука `192.168.1.42`, порт `5002`
4. Узнайте внешний IP: https://ifconfig.me
5. Откройте: `http://ВАШ_ВНЕШНИЙ_IP:5002`

> ⚠️ **Безопасность:** При доступе через Интернет рекомендуется использовать ngrok (HTTPS), а не открытый проброс портов.

### Firewall (Брандмауэр)

Если с другого устройства страница не открывается, разрешите порт в брандмауэре:

**Windows:**
```powershell
netsh advfirewall firewall add rule name="DRGR Server" dir=in action=allow protocol=tcp localport=5002
```

**Linux:**
```bash
sudo ufw allow 5002/tcp
```

## ❓ Решение проблем

**Порт 5002 занят**
→ Запустите: `$env:DRGR_PORT=5002; python vm/server.py`
→ Или используйте `start.ps1` — он автоматически найдёт свободный порт.

**"Python не найден"**
→ Установите Python с https://www.python.org/downloads/ (отметьте "Add Python to PATH").

**Бот не отвечает в Telegram**
→ Проверьте BOT_TOKEN в настройках (http://localhost:5002 → Настройки).

**Нет AI (чат/статьи не работают)**
→ Установите Ollama: https://ollama.com → `ollama pull llama3`

**VM-сервер недоступен (http://localhost:5002)**
→ Запустите `start.ps1` или `ЗАПУСТИТЬ_БОТА.bat`.

## 🧠 Модуль психодиагностики (Psycho-Platform)

Модуль для психоэмоциональной оценки на основе распознавания эмоций по лицу (FER), анализа зрачков и текстовых тестов.

- **FER-анализ** — анализ лицевой экспрессии во время тестов (валентность, возбуждение, стресс)
- **🔍 Анализ зрачков** — дилатация, анизокория → стресс / когнитивная нагрузка / расслабление
- **👁️ Возраст по глазам** — 5 параметров (морщины, мешки, склера, радужка, птоз) → оценка возраста
- **📊 Совокупная оценка** — FER (40%) + зрачки (30%) + baseline (30%) → единый стресс-балл + рекомендации
- **21+ Gate** — проверка возраста по лицу (вероятностная оценка)
- **Этика** — обязательное информированное согласие, без диагнозов, только рекомендации
